CodeLab - HTML

About CodeLab

CodeLab is an interactive learning platform that allows users to experiment with code in a live environment. This version focuses on HTML, offering features like syntax highlighting and code editing in a clean, modern UI.

Description

Developed with TailwindCSS and CodeMirror, CodeLab gives you a seamless experience for coding and learning web development. Explore different coding languages and test your skills right here. The platform is designed to be simple yet powerful, perfect for both beginners and advanced users.

Shortcuts

Windows Shortcuts

  • Ctrl + S - Save and Run
  • Ctrl + Enter - Save and Run
  • Ctrl + / - Comment
  • Ctrl + B - Settings

Mac Shortcuts

  • Cmd + S - Save and Run
  • Cmd + Enter - Save and Run
  • Cmd + / - Comment
  • Cmd + B - Settings

FAQ

How do I save my project?

+

You can save your work by using the "Ctrl + S" shortcut or by clicking the save button in the editor.

Issue with <script> tag?

+

Place a space between the script tag like: <script> </script>

Developer Features

+

You can create multiple files at once, for example: about.html, contact.html, app.js.
To create new files, use the "Create Files" button in the editor toolbar.
Click to Import files. Accepts .html, .css, or .js files.
After importing, the file will be available for editing in your project.

How to connect .json files?

+

To connect the .json files you can simply place data=window['data.json'].
