HTML Tags for Beginners

Learn the most common HTML tags with examples and explanations!

1. The <h1> to <h6> Tags (Headings)

The <h1> to <h6> tags are used to define headings in HTML. <h1> is the largest, and <h6> is the smallest.

<h1>This is a Heading 1</h1>

This is a Heading 1

2. The <p> Tag (Paragraph)

The <p> tag is used to define a paragraph of text. It automatically adds space before and after the text.

<p>This is a paragraph.</p>

This is a paragraph.

3. The <a> Tag (Link)

The <a> tag is used to create links. You can link to other pages or websites by using the href attribute.

<a href="https://www.example.com">Visit Example</a>
Visit Codelab - Html Editor

4. The <img> Tag (Image)

The <img> tag is used to display images on a webpage. It uses the src attribute to specify the image source and alt for alternative text.

<img src="https://cdn.pixabay.com/photo/2017/06/19/04/31/logo-2418159_1280.png" alt="A description of the image">

5. The <ul> and <ol> Tags (Lists)

The <ul> tag is used to define an unordered list, and the <ol> tag is for ordered lists. Both use the <li> tag for each list item.

Unordered List:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3

Ordered List:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>
  1. First item
  2. Second item
  3. Third item

6. The <div> Tag (Division)

The <div> tag is used to group content together. It doesn't add any styling by itself but is useful for applying CSS styles or JavaScript.

<div class="container">Content here</div>
Content here

7. The <span> Tag (Inline Element)

The <span> tag is used for inline grouping. It doesn't add any layout effect but can be styled using CSS.

<span style="color: red;">This is red text</span>
This is red text

8. The <form> Tag (Form)

The <form> tag is used to create a form for user input. It can contain different input elements like text fields, checkboxes, and buttons.

<form>
    <label for="name">Name</label>
    <input type="text" id="name">
    <button type="submit">Submit</button>
</form>

9. The <br> Tag (Line Break)

The <br> tag is used to create a line break, so text or other elements appear on the next line. 

<p>This is text<br>This is on a new line</p>

This is text
This is on a new line

10. The <strong> and <em> Tags (Text Formatting)

The <strong> tag is used to define strong emphasis, usually resulting in bold text, while the <em> tag is used for emphasizing text, usually rendering it in italics.

<strong>Bold Text</strong> <em>Italic Text</em>
Bold Text Italic Text
